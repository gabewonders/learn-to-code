Comparators

Comparators allow us to determine if something is true or false.

  1. == equals
  2. != not equal
  3. < less than
  4. <= less-than-equal
  5. > greater than
  6. >= greater-than-equal
6 == 6 #returns True
7 != 4 #returns True
3 > 9 #returns False
2 < 4 #returns True
6 <= 6 #returns True
7 >= 9 #returns False

Note that = is different than ==. = assigns. Whereas == compares.

Boolean Operators

Sometimes you want to combine comparators together to form more complex logic. This is what boolean operators are for.

and checks if both statements are true
or checks if only one statement is true
not is the opposite of a statement

2 < 4 and 3 > 9 #returns False
7 == 13 or 4 > 1 #returns True
not 4 == 5 #returns True

Order of operations!

Ah not math! Yes, there is an order of operations in the way these boolean operators are evaluated. It is important that you know this so you aren’t surprised by your computers answers.

  1. not is evaluated first
  2. and is evaluated next
  3. or is evaluated last

For example:

#This returns True
bool_example = True or not False and False

Control Flow

Using comparators we can now begin to build logic into our programs. This will let our code decide to go in one direction or another based on the results of our comparisons.

if statement

if is a conditional statement that executes some specific code after checking if its expression is True

if 3 < 8:
    print "Three is less than eight!"

If our test condition had been False nothing would have been displayed because our program would have jumped past all of the code inside the if block. Enter else statements.

Notice the colon and the indentation after the if statement. These aren’t optional! This is how Python knows that we are still inside of the if statement. As soon as Python sees a line that isn’t indented, that is the end of the if statement. Same goes for functions.

Whitespace

Whitespace is used to structure code. Python focuses on readability.

#this code will cause an indentation error

print "I have oil drums! Time to sell them"

You will get this error every time your whitespace is off. So, lets examine this error.

Indent your code with 4 spaces

  oil_drum = 12
  return oil_drum

else statement

The else statement is the second keyword that we can use in combination with the if keyword. An if/else pair says: “If this expression is true, run this indented code block; otherwise run this code after the else statement.”

Unlike if, else doesn’t depend on an expression, it is a catch-all. For example:

if 3 > 8:
    print "I don't get printed!"
else:
    print "I get printed!"

elif statement

elif is short for “else if” and can be used to add more possible options after an if statement. 

num = 15
if num < 10:
    print "number is less than 10"
elif num > 10:
    print "number is greater than 10"
else:
    print "number is equal to 10"

Functions

A function is a collection of code that we group together and give a name. We can now use this group of code as many times as we want. 

def square(number):
    """Squares the number passed to the function"""
    sqr_num = number * number
    return sqr_num

square(5) #returns the value 25

Function Syntax

  1. The header, which includes the keyword def (stands for define), the name of the function, parenthesis () and any parameters the function requires.
    def hello_world(): // There are no parameters

  2. An optional comment explaining what the function does.
    """Prints 'Hello World!' to the console"""

  3. The body, which describes the procedures the function carries out. The body is indented, just like for conditional statements.
    print "Hello World!"

Putting it all together the function it looks like this:

def hello_world(): 
    """Prints 'Hello World!' to the console"""
    print "hello world"
